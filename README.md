# Durgaprasad B - Personal Portfolio

Welcome to my modern personal portfolio, built with Ruby on Rails! This project serves as a showcase of my skills in backend development, deployment, and modern web design, with a focus on Cloud Computing and Software Engineering.

## Features

- **Modern UI/UX**: Features a sleek, dark-themed Bento Grid layout built with modern CSS (Flexbox/Grid, CSS Variables, and custom animations).
- **Responsive Design**: The layout adapts seamlessly across all device sizes.
- **Project Showcase**: Highlights key projects like E-Commerce Analytics Pipeline, Automated Trading Bots, and other major work.
- **Ruby on Rails 8**: Built using the latest Ruby on Rails framework, taking advantage of modern setups for performance and simplicity.

## Tech Stack

- **Backend Framework**: Ruby on Rails 8.x
- **Language**: Ruby 3.x
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla with CSS Grid/Flexbox)
- **Database**: SQLite3 (default for initial development)

## Getting Started

To run this application locally, ensure you have Ruby and Rails installed on your system.

### Prerequisites

- Ruby version `3.3.x` or higher
- Bundler

### Setup Instructions

1. **Clone the repository** (if pushed to GitHub):
   ```bash
   git clone <repository-url>
   cd "portfolio(Ruby on Rails)"
   ```

2. **Install dependencies**:
   Run Bundler to install the required gems.
   ```bash
   bundle install
   ```

3. **Start the Rails server**:
   Launch the development server.
   ```bash
   ruby bin\rails server
   ```

4. **View the Application**:
   Open your preferred web browser and navigate to `http://localhost:3000`.
